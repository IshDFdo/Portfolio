# Modern Frontend Developer Portfolio

This is a ready-to-edit personal portfolio website for a frontend web developer in Sri Lanka.

## Files
- `index.html` - website structure
- `style.css` - full design and responsive styling
- `script.js` - mobile menu interaction

## How to edit
1. Open `index.html`.
2. Change the name, email, social media links and project links.
3. Replace `https://example.com` in the project cards with your real project URLs.
4. Replace `yourmail@example.com` with your real email address.
5. Open `index.html` in a browser to view the website.
